Welcome to StEVN

StEVN tells stories.

StEVN (Story Engine for Visual Novels) is a WIP engine for visual novels and light adventure games which are similar to the Ace Attorney games created by Capcom.

StEVN utilises the libgdx game development framework.

This project was started to support the author's World Beyond The Wall adventure game project, and to practice implementing good programming habits in Java. The goal of this project is a simple, easy-to-understand engine for games.

StEVN is simple.

StEVN uses Java.
